# The Agency Programming Language

Welcome! This book will teach you all the major features of the Agency language, starting with the basics like syntax and how to execute an Agency program, and building up to more powerful features.

A quick note before you start: since syntax highlighting libraries don't know about Agency, all the Agency code blocks are set to have the TypeScript as the language in order to get syntax highlighting. Just a heads up so you don't get confused if you see `ts` in the code block.

Also: The Agency language compiles to and can interoperate with either TypeScript or JavaScript. I'll just say TypeScript everywhere, because writing "TypeScript/JavaScript" gets really boring, but you'll know what I mean.
